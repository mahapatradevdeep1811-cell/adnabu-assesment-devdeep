Assumptions (Automation)
=======================

While writing the automation script, I made a few assumptions based on how the website behaves and what was required in the assignment.

- I assumed that the store will always ask for a password before accessing the site, so I included that step in the script.

- I assumed that the search functionality is working and will return results for a valid keyword like "iPhone".

- I assumed that at least one product will be available in the search results, so the script selects the first product.

- I assumed that the first product selected is available for purchase (i.e., not out of stock).

- I assumed that clicking the "Add to Cart" button will successfully add the product without requiring any additional selections (like size or color).

- I assumed that once a product is added, it will be visible on the cart page and can be verified by checking if items are present.

- I assumed that the page elements (like search icon, input field, and buttons) are stable and do not change frequently.

- Since this is a basic automation task, I focused only on one main scenario and did not handle advanced cases like multiple products, filters, or checkout.

- I also assumed that the execution environment (browser and driver) is properly set up before running the script.
