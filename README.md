# AdNabu QA Assignment
==========================

## Overview
----------------
This repository contains test cases and bug reports for AdNabuTestStore, an e-commerce web application.

## Scope
----------
The following features were tested:

* Product Search
* Add to Cart

## Test Approach
--------------------
The focus was on covering key user flows and ensuring different types of scenarios were included:

* Positive cases to verify expected behavior
* Negative cases to check how the system handles invalid inputs
* Edge cases to test boundary conditions

## Test Data
--------------
Sample inputs such as "iPhone", partial keywords, invalid inputs, and empty values were used to validate search behavior.

## Project Structure
-----------------------
* test-cases/test_cases.md → Contains all test cases
* notes/bug_reports.md → Contains identified bugs
* notes/assumptions.md → Lists assumptions made during testing

## Test Environment
----------------------
* Website: https://adnabu-store-assignment1.myshopify.com
* Store Password: AdNabuQA
* Browser: Chrome

----------------
